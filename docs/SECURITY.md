# Security Policy

## Supported Versions

| Version | Supported          |
|---------|--------------------|
| 1.x     | :white_check_mark: |
| 0.x     | :x:                |

> Only the latest version receives security updates.

## Reporting a Vulnerability

If you discover a vulnerability, please:

1. Do not open a public issue.
2. Email: security@arc-runner.dev
3. Provide enough details to reproduce the issue.

## Triage Process

- You will receive a confirmation within 2 business days.
- The team will investigate and respond within 7 business days with an initial status.
- If confirmed, we will work on a fix and coordinate responsible disclosure.

Thank you for helping keep ARC Runner secure!
